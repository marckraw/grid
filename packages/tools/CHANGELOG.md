# @mrck-labs/grid-tools

## 0.1.0

### Minor Changes

- Initial release of @mrck-labs/grid-tools package
- Extracted tools from @mrck-labs/grid-core for better modularity
- Includes ten ready-to-use tools:
  - `calculatorTool` - Basic math operations
  - `currentTimeTool` - Get current date and time with timezone support
  - `readUrlTool` - Scrape and read URL content using Firecrawl
  - `randomNumberTool` - Generate random numbers within a range
  - `stringUtilsTool` - String operations (reverse, count, base64, case conversion)
  - `jsonFormatterTool` - Format, validate, and minify JSON
  - `systemInfoTool` - Get OS, memory, CPU, and environment information
  - `delayTool` - Async delay operation for timing needs
  - `hashTool` - Generate MD5, SHA1, SHA256, SHA512 hashes
  - `dataConverterTool` - Convert between JSON, CSV, and TSV formats
